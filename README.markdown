======================================

Editor
--------------------------------------
* Main     : MacVim
* Plugin   : [janus](https://github.com/carlhuda/janus)
* Install  : cd ~/.vim; rake

Shell
--------------------------------------
* Main     : Zsh
* Plugin   : [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)
